SYNTAX: radial-gradient ( shape size at position, color-stop1, color-stop2,....!);

the agrument of the radial-gradient() function has the following meaning:

  1. position- specifies the starting point of the gradient ending shape. it can be circle or ellipse
  2. position- specifies the starting point of the gradient ending shape. it can be circle or ellipse
