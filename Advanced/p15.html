CSS Pseudo-Classes...!

The CSS Pseudo-Classes allow you to style te dynamic states of an elements such as hover, active and focus state, as well as elements that are existing in the document tree but cant be targeted via the use of other selectors without adding any IDs or classes to them, for example, targeting the first or last child elements

A Pseudo-Classes starts with a colon (:) Its syntax can be given with:

Selector:pseudo-class { property: value; }
